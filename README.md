**oslab** is a toolchain image for operating system course in ZJU.

## Usage

```bash
$ docker pull alphavake/oslab
```

## Toolchain List

- **qemu-system-riscv64**: v6.1.0
- **riscv-gnu-toolchain**: nightly 2021.09.21
